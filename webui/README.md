# gcsim WebUI

原神シミュレーター用のWebインターフェース

## 概要

gcsim WebUIは、GCSLコンフィグファイルを使用してシミュレーションを実行し、結果を視覚的に表示するWebアプリケーションです。

## 特徴

- **日本語インターフェース**: すべてのUIが日本語で表示されます
- **シンタックスハイライト**: GCSLコンフィグのための読みやすいコードエディタ
- **リアルタイム結果表示**: シミュレーション結果を即座に表示
- **統計情報**: DPS、EPS、RPS、HPS、SHP、期間などの主要統計値を表示
- **キャラクター情報**: 各キャラクターのステータス、武器、聖遺物情報を表示
- **ターゲット情報**: 敵のレベル、HP、元素耐性などを表示
- **グラフ表示**: DPS分析、ダメージ分布、エネルギー、元素反応などのグラフ
- **マテリアルデザイン**: モダンで読みやすいUIデザイン

## 使い方

### サーバーの起動

```bash
./gcsim-webui -addr=:8383
```

デフォルトでは `:8382` ポートでリッスンしますが、`-addr` フラグでポートを変更できます。

### ブラウザでアクセス

```
http://localhost:8383/ui/index.html
```

### シミュレーションの実行

1. テキストエリアにGCSLフォーマットのコンフィグを入力
2. 「シミュレーション実行」ボタンをクリック
3. 結果が下部に表示されます

## API

### POST /api/simulate

シミュレーションを実行するAPIエンドポイント

**リクエスト:**
```json
{
  "config": "GCSLコンフィグテキスト"
}
```

**レスポンス:**
シミュレーション結果のJSON（詳細は `pkg/model` を参照）

## 技術スタック

- **フロントエンド**: Pure HTML/CSS/JavaScript（外部依存なし）
- **バックエンド**: Go + `pkg/simulator`
- **グラフ**: カスタムCanvas実装

## 注意事項

- このWebUIは、既存の `serve`/`server` モードとは完全に独立した機能です
- シミュレーションのタイムアウトは180秒に設定されています
- ブラウザで実行されるため、大規模なシミュレーション（反復回数が多い）は時間がかかる場合があります

## ファイル構成

```
webui/
├── index.html    # メインHTMLファイル
├── app.js        # JavaScriptロジック
└── README.md     # このファイル
```

## 開発

WebUIの開発を行う場合:

1. `webui/` ディレクトリでファイルを編集
2. サーバーを再起動
3. ブラウザをリロード

変更はすぐに反映されます（サーバー再起動不要でHTMLとJSの変更が有効になります）。
